# MyPersonal

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Aplicações Web`

`SEMESTRE: 1º`

O projeto consiste no desenvolvimento de uma plataforma web front-end voltada para a centralização do acompanhamento entre profissionais da saúde (como personal trainers e nutricionistas) e seus respectivos alunos ou clientes. O objetivo principal é integrar, em um ambiente único e acessível, informações que atualmente costumam ficar fragmentadas em diversos canais, tais como agendas, planos de treino, dietas, vídeos de apoio e registros de evolução física. A solução busca otimizar a produtividade do profissional ao reduzir o retrabalho e, simultaneamente, melhorar a experiência e a adesão do aluno por meio de uma interface intuitiva e responsiva.

A proposta foca especialmente na experiência do usuário (UX) e na organização visual, atendendo desde profissionais que gerenciam grandes carteiras de clientes até alunos iniciantes que necessitam de orientações claras e linguagem simples. Embora o escopo inicial priorize as funcionalidades de interface e a navegação fluida em dispositivos móveis e desktop, o projeto estabelece uma base para futuras expansões, mantendo o foco em requisitos de acessibilidade, segurança de dados e usabilidade minimalista.
